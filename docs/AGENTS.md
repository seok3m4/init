# Docs AGENTS

문서 폴더는 PM이 1차 소유하고, 변경되는 도메인의 개발자가 함께 리뷰한다.

## Rules

- 계약 변경은 `docs/03_contracts`를 먼저 수정한다.
- 데이터 모델, 상태 전이, 비동기 파이프라인 변경은 `docs/02_architecture`를 함께 수정한다.
- 화면/사용자 흐름 변경은 `docs/01_product`를 함께 수정한다.
- 분업, 마일스톤, 테스트 전략 변경은 `docs/04_implementation`을 수정한다.
- AI 작업자 라우팅 변경은 `docs/05_agents/AGENTS.md`를 수정한다.

## Review Owners

- Auth/Infra 문서: A
- Company Recruiting 문서: B
- Company Interview/Criteria 문서: C
- Candidate/Application/Interview 문서: D
- AI Report/Pipeline 문서: E
- 발표/검증 문서: PM
